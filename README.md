## Ar

اهلا

باللغة العربية تعني "ورقة غش"

هي مذكرة ملخص فيها الموضوع المذكور فيها

فائدتها؟ كبشر تمر علينا الكثير من البرامج و الأدوات ولكن بعض المرات ننسى طريقة تشغيلها او استخدامها وهذا شي طبيعي

فهذي الورقة تكون مرجع بسيط و سريع

## طريقة الإستخدام

- كل مجلد يحمل اسم برنامج
  
- داخل كل مجلد يوجد ملفين احدهم بالعربية و احدهم بالإنقليزية
